# Transformer_Power_Supply
This project is a transformer-based power supply that converts high-voltage AC into low-voltage DC. It uses a step-down transformer to reduce the voltage, followed by a bridge rectifier to convert AC to DC, and a filter capacitor to smooth the output. This design helped me understand the basics of power supply circuits, including voltage conversion, rectification, and filtering. It is a simple and reliable circuit, useful for powering low-voltage electronic devices.

Here are the key points:
Converts high-voltage AC to low-voltage DC.
Uses a step-down transformer for voltage reduction.
Bridge rectifier (4 diodes) converts AC to pulsating DC.
Filter capacitor smooths the outpuy.
Can include voltage regulator for stable output.
Simple and reliable power supply circuit.
Useful for low-voltage electronic devices.
Helps understand rectification, filtering, and voltage conversion.
